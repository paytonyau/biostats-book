# Images Directory

Place your figure files here. Each chapter references one or more images using the `{figure}` directive in MyST Markdown.

## Recommended tools for creating these figures

- **R + ggplot2** — professional-quality plots, fully reproducible
- **Canva / draw.io** — conceptual diagrams (nested layers, flowcharts, target diagrams)
- **BioRender** — if your institution has access, good for clinical illustrations

All images should be exported at **300 dpi** minimum, **PNG format**, with a white background.
