## Dataset Attribution

The worked examples throughout this book use **Dr John Snow's 1854 Broad Street cholera outbreak dataset**, distributed as `Snow.deaths` in the R package `HistData`:

> Friendly, M. (2022). *HistData: Data Sets from the History of Statistics and Data Visualization*. R package version 0.8-7. [https://CRAN.R-project.org/package=HistData](https://CRAN.R-project.org/package=HistData)

The original data were compiled from:

> Snow, J. (1855). *On the Mode of Communication of Cholera* (2nd ed.). John Churchill, London.
